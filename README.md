# 🌫️ Misty Village Chronicles — Frontend

마을 관리 RPG 게임의 프론트엔드 레포지토리입니다.  
백엔드 API와 통신하여 게임 UI를 렌더링합니다.

---

## 🛠 기술 스택

| 항목 | 내용 |
|------|------|
| Framework | Next.js 16.2.3 |
| UI Library | React 19 |
| Styling | Tailwind CSS v4 |
| Language | JavaScript / TypeScript |

---

## 📁 프로젝트 구조

```
misty-village-front/
├── app/            # Next.js App Router 기반 페이지
├── components/     # 재사용 UI 컴포넌트
├── lib/            # 유틸리티 / API 호출 함수
├── public/         # 정적 파일 (이미지, 아이콘 등)
└── package.json
```

---

## ⚙️ 로컬 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (http://localhost:3000)
npm run dev
```

> 백엔드 서버(`misty-village-backend`)가 먼저 실행되어 있어야 합니다.
